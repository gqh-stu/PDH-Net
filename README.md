# PDH-Net

**Official PyTorch implementation of the paper: "PDH-Net: A Parallel Dual-branch Hybrid Network for Camera-based Blood Oxygen Saturation Measurement"**

> **🚨 Status Update:**
> The source code of PDH-Net is currently deeply integrated with several ongoing, unpublished follow-up projects in our laboratory. To protect these unreleased works, we are currently decoupling the framework to prepare a standalone version. 
> 
> The complete standalone code, data processing scripts, and pre-trained models will be made publicly available in this repository immediately upon the acceptance and publication of the manuscript. 

## Overview
Blood oxygen saturation ($SpO_2$) serves as a critical parameter for evaluating cardiopulmonary function. This repository will contain the implementation of **PDH-Net**, a novel Parallel Dual-branch Hybrid Network designed for robust, non-contact $SpO_2$ estimation from facial videos. 

Our architecture fuses:
* An **Attention-Enhanced Convolutional Neural Network (AE-CNN)** branch to capture fine-grained local features.
* An **Enhanced Hierarchical Video Vision Transformer (EH-ViViT)** branch to model global spatio-temporal context.
